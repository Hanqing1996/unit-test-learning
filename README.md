#### 为什么单元测试在前端难以推广
1. 需求变更快
2. 框架迭代快

#### 单元测试有什么用
1. 除了前端，其他语言都重视单元测试
2. 开发库（轮子）必须要有
3. 业务代码不需要单元测试
